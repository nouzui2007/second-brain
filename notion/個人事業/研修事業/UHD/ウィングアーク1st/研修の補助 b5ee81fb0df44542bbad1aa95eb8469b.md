## PostgreSQLインストール

```bash
# インストール
sudo yum install postgresql-server postgresql-devel postgresql-contrib
# 初期化
sudo su - postgres
postgresql-setup initdb
# 起動
sudo systemctl start postgresql.service
# 自動起動設定
sudo systemctl enable postgresql.service
# ステータス確認
sudo systemctl status postgresql.service
```

### postgresユーザ以外からログインを許可する

参考サイト：

https://www.koikikukan.com/archives/2020/06/03-235555.php

- テキストを編集

```bash
sudo vi /var/lib/pgsql/data/pg_hba.conf
```

- localのpeer → trust、host(127.0.0.1/32)のident → trust

```bash
# TYPE  DATABASE        USER            ADDRESS                 METHOD
 
# "local" is for Unix domain socket connections only
local   all             all                                     peer
# IPv4 local connections:
host    all             all             127.0.0.1/32            ident
# IPv6 local connections:
host    all             all             ::1/128                 ident
```

```bash
# TYPE  DATABASE        USER            ADDRESS                 METHOD
 
# "local" is for Unix domain socket connections only
local   all             all                                     trust
# IPv4 local connections:
host    all             all             127.0.0.1/32            trust
# IPv6 local connections:
host    all             all             ::1/128                 ident
```

- 再起動

```bash
sudo systemctl restart postgresql.service
```

## PHPインストール

```bash
# インストール
sudo amazon-linux-extras install php7.2 -y
sudo yum install php-pgsql
# 起動
sudo systemctl start php-fpm.service
# 自動起動
sudo systemctl enable php-fpm.service
```

### nginxとphp-fpmのUNIXドメインソケット接続確認

```bash
netstat -al --protocol=unix |egrep "Proto|fpm"
```

### php-fpmの設定

```jsx
sudo vi /etc/php-fpm.d/www.conf
```

apache → nginx に修正

### 接続確認用phpファイル追加

```bash
sudo echo '<?php phpinfo(); ?>' > /usr/share/nginx/html/phpinfo.php
```

### 再起動

```bash
sudo systemctl restart php-fpm.service
sudo systemctl restart nginx.service
```

### 確認

http://[ホストのIPアドレス]/phpinfo.php

ページが表示されることを確認する。
