# HTML

# Stoplight公式の方法でHTML表示する

## HTMLの用意

yamlファイルを読み込んで表示するHTMLを作成する。

**elements-api**タグの**apiDescriptionUrl**属性にyamlファイルを指定する。

ファイル名をindex.htmlにしておくと便利。

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
      name="viewport"
      content="width=device-width, initial-scale=1, shrink-to-fit=no"
    />
    <title>Elements in HTML</title>

    <script src="https://unpkg.com/@stoplight/elements/web-components.min.js"></script>
    <link
      rel="stylesheet"
      href="https://unpkg.com/@stoplight/elements/styles.min.css"
    />
  </head>
  <body>
    <elements-api apiDescriptionUrl="./jrshikoku-ticketless.yaml" router="hash" />
  </body>
</html>
```

## Webサーバ起動

ローカルファイルでの表示ができないので、nginxコンテナを起動して表示する。

htmlがあるフォルダをnginxコンテナのドキュメントルートにマッピングする。

```html
docker run --rm -v "$(pwd)":/usr/share/nginx/html -p 8080:80 nginx
```

## 確認

ローカルホスト、ポートマッピングしたポートにアクセスすると表示される。

```html
http://localhost:8080/
```

# RedocでYAMLをHTML表示する

## Redoc公式コンテナを起動

Redoc公式イメージでHTML表示する。

yamlファイルがあるディレクトリで以下のコマンドを実行する。

SPEC_URLにHTML表示したい対象のyamlファイルを指定する。

```
docker run --rm -p 8080:80 -v "$(pwd)":/usr/share/nginx/html/file -e SPEC_URL=file/jrshikoku-ticketless.yaml redocly/redoc
```

## 確認

ローカルホスト、ポートマッピングしたポートにアクセスすると表示される。

```html
http://localhost:8080/
```

# Redoc-cliでHTMLを生成する

## インストール

npmでインストールする。

### npmインストール

1. nodebrewをインストールする。nodebrewはnode.jsのバージョンを管理するツール。
    
    ```html
    brew install nodebrew
    ```
    
2. パスを通す。
    
    ```html
    echo 'export PATH=$HOME/.nodebrew/current/bin:$PATH' >> ~/.zshrc
    source ~/.zshrc
    ```
    
3. node.jsをインストールする。
    
    ```html
    nodebrew install-binary latest
    
    # 安定版のインストールの場合はこれ
    nodebrew install-binary stable
    ```
    
    エラーが発生する時があるが、エラー時はソースディレクトリを作成して、再度インストールを実行する。
    
    ```html
    mkdir -p ~/.nodebrew/src
    ```
    
4. 利用するnode.jsのバージョンを指定する。
    
    ```html
    # 利用可能なバージョンを確認
    nodebrew list
    
    # 利用するバージョンを指定
    nodebrew use v18.12.1
    ```
    
5. npmが使用できるか確認する。
    
    ```html
    npm -v
    ```
    

### Redoc-cliインストール

```html
npm install -g redoc-cli
```

## コンバート

yamlファイルがあるディレクトリで以下のコマンドを実行する。

```markdown
redoc-cli bundle jrshikoku-ticketless.yaml -o index.html
```

## 確認

htmlファイルをダブルクリックでブラウザ確認する。
