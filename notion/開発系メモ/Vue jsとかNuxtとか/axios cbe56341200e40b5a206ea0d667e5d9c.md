# axios

## $付きとそうでないもの

`@nuxt/axios` で用意されたもので、本家と同名のものが存在する場合に **$** をつけている

## $axiosとaxios

別物なので注意

`axios` には `onRequest` は用意されていないので、利用する際も動作しない

そのため、pluginsで `$axios` に `onRequest` を設定しても、 `import axios from "axios"` でインポートした `axios` とは別物らしく、 `onRequest` が働かない

**nuxt** からは `this.$axios` でアクセスできるが、勝手に定義した関数の `this` ではアクセスできない

## $getとget

`$axios.get` で取れるもの

```jsx
response
　 |- data レスポンスデータ
　 |- status ステータスコード
　 |- statusText ステータステキスト
　 |- headers レスポンス
　 |- config コンフィグ
```

`$axios.$get` はレスポンスデータのみ
