# Vuex

## Store下のファイル

ファイル名がネームスペース的な働きをしている

### アクセス方法

**getters**

```jsx
store.getters["ファイル名/getter名"]
```

**actions**

```jsx
this.$store.dispatch("ファイル名/アクション名", パラメータ)
```
