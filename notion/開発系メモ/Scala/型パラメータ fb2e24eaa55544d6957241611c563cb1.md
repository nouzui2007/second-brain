# 型パラメータ

Javaで言うジェネリクス

クラスの定義では0個以上の型をパラメータとして受け取れる

型パラメータはクラス名の右側に[]で指定する

## 型を指定しないで、値を保持するクラス

```scala
class Cell[T](var value: T) {
  def put(newValue: T) = {
    value = newValue
  }
  
  def get(): T = value
}

val cell = new Cell[Int](1)
cell.put(10)
println(cell.get())

// newの時、クラスのところの型は省略可能
val cell2 = new Cell("String Context")
println(cell2.get())
cell2.put("Change String")
println(cell2.get())
```

## 実践的な例

割り算の答えと余りを返す用のクラスを作ってみる

```scala
class Pair[A, B](val a: A, val b: B) {
  override def toString() = s"($a, $b)"
}

def divide(m: Int, n: Int) = new Pair[Int, Int](m / n, m %n)

println(divide(7, 3))
```
