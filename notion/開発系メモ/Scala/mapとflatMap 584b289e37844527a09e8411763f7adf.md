# mapとflatMap

配列やリストに対して繰り返し処理を行い、新たな配列やリストを作成する。

OptionやFutureなどの型から値を取り出す。

# OptionやFutureから値を取り出す場合の違い

Option型の変数で `map` を実行する。

```scala
val s = Some("a")
s.map(m => m.length)
```

このとき、map内の `m` は `String` で、戻り値は `Int` である。mapの戻り値は `Option` 型になる。

一方 `flatMap` の場合は、flatMap内の `m` は `String` であるものの、戻り値は `Option` 型であることが期待されている。そのため `m` を返すとエラーになる。

```scala
val s = Some("a")
s.flatMap(m => m.length) // エラーになる
s.flatMap(m => Some(m.length)) // こうしないといけない
```
