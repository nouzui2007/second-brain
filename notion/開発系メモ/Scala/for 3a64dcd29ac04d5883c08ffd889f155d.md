# for

```scala
for (x <- 1 to 9; y <- 1 until 10) {
  println(s"$x x $y = ${x * y}")
}

val l = for {
  x <- 1 to 9
  y <- 1 until 10
} yield s"$x x $y = ${x * y}"

l.foreach(println)
```

### ２つの値を使ってタプルにする

```scala
val s1 = Some("hogehoge")
val s2 = Some("mogemoge")

for {
  h <- s1
  m <- s2
} yield (h, m)
```

## Try型を返す

オブジェクトを作成して、それを検証する。

`moge` メソッドの結果がSuccessだったら、 `for` の戻り値は `hoge` メソッドの戻り値になる。

`moge` メソッドの結果がFailureだったら、 `for` の戻り値はFailureになる。

```scala
def hoge() = Success("Created Object")
def moge() = Failure(new Exception("Failure Validation"))
// def moge() = Success("Success Validation")

for {
  h <- hoge()
  _ <- moge()
} yield h
```
