# flatMap

mapとflattenを行う

mapのあとflattenを行う

引き取る関数は文脈付きで返す必要あり

```scala
val option: Option[Int] = Some(100)
val result: Option[Int] = option.flatMap { value => Some(value + 120) }
println(result)

val option2: Option[Int] = None
val result2: Option[Int] = option2.flatMap { value => Some(value + 120) }
println(result2)

val list: List[Int] = List(1, 10, 100)
val result3: List[Int] = list.flatMap { value => // <- ふつうの値をとって
  List(value, value + 1, value + 2) // <- 文脈付きの値で返す
}
println(result3) // -> List(1, 2, 3, 10, 11, 12, 100, 101, 102)

val list2: List[Int] = Nil
val result4: List[Int] = list2.flatMap { value => 
  List(value, value + 1, value + 2) 
}
println(result4) // -> List(1, 2, 3, 10, 11, 12, 100, 101, 102)

List(List(1, 2), List(3, 4, 5), List(6), List(7, 8)).flatMap { m => { m.map { m2 => m2}} }
```
