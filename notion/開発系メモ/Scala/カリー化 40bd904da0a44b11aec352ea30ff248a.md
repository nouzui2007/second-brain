# カリー化

## 通常の関数

```scala
val add = (x: Int, y: Int) => x + y
println(add(1, 3))
```

## カリー化の実装

２つある引数を一つ減らして、もう一つの引数を持つ関数を返す

```scala
// 一般的なカリー化
val curried = (x: Int) => (y: Int) => x + y
println(curried(1)(3))

// Scalaでの簡単な宣言
def curried2(x: Int)(y: Int) = x + y
println(curried2(1)(3))

println(add.curried(1)(2))
```

カウンターをカリー化で実装してみる

```scala
def countFunction(start: Int): () => Int = {
  var i = start
  () => {i += 1; i}
}
val counter = countFunction(2)
for (i <- 1 to 5) {
  println(counter())  
}
```
